# Smile Right Dental Clinic Website

Professional website for Dr. Sneha Kedia's dental practice in Kandivali East, Mumbai.

## Features

- **SEO Optimized**: Comprehensive meta tags, Open Graph tags, and structured data (Schema.org)
- **Responsive Design**: Mobile-first approach, works seamlessly on all devices
- **Professional Theme**: Navy blue and orange color scheme matching the clinic's branding
- **Fast Loading**: Optimized code with minimal dependencies
- **Accessibility**: Semantic HTML and ARIA labels where needed
- **Local SEO**: Schema markup for local business, Google Maps integration

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Poppins)

## Files Structure

```
smile-right/
├── index.html          # Main HTML file
├── styles.css          # Stylesheet with responsive design
├── script.js           # JavaScript for interactivity
├── robots.txt          # Search engine crawling instructions
├── sitemap.xml         # XML sitemap for SEO
├── images/             # Clinic photos and logo
└── README.md          # This file
```

## Setup Instructions

1. Open `index.html` in any modern web browser
2. For deployment, upload all files to your web hosting service
3. Update the domain name in `sitemap.xml` and `robots.txt` to match your actual domain

## Customization

To update the domain name throughout the site:
1. Replace `https://www.smilerightdental.org/` in `sitemap.xml` with your actual domain
2. Update the same in `robots.txt`
3. Add your actual Google Maps embed URL in `index.html`
