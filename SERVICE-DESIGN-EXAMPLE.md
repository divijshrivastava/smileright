## Animation Details

### Expand Animation
```css
@keyframes slideDown {
  from {
    opacity: 0;
    transform: translateY(-20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

### Hover Effects
```
Gallery Image:
  Normal: scale(1.0)
  Hover:  scale(1.0) translateY(-4px) + shadow

Expand Icon:
  Collapsed: rotate(0deg)
  Expanded:  rotate(180deg)
```
